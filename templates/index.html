      Popüler Minecraft Komutları ve Örnekler

      Aşağıda her sürüm grubu için en çok kullanılan Minecraft komutları ve açıklamaları bulunmaktadır. Bu komutları doğrudan kopyalayıp oyununuzda kullanabilirsiniz.

      Give Komutları

      1.21 Components: Item verme komutu artık NBT yerine component sistemi kullanır.

      /give @s diamond_sword[
  custom_name='Ejderha Kılıcı',
  enchantments={
    levels:{sharpness:10,fire_aspect:3}
  },
  unbreakable={}
]

      1.20.4 Modern:

      /give @s diamond_sword{
  display:{Name:'{"text":"Ejderha Kılıcı"}'},
  Enchantments:[
    {id:sharpness,lvl:10},
    {id:fire_aspect,lvl:3}
  ],
  Unbreakable:1
}

      1.12.2 Legacy:

      /give @s minecraft:diamond_sword 1 0 {
  display:{Name:Ejderha Kılıcı},
  ench:[
    {id:16,lvl:10},
    {id:20,lvl:3}
  ]
}

      Minecraft Komut Türleri ve Kullanım Alanları

      Item Komutları

      Minecraft'ta özel itemlar oluşturmak için /give komutu kullanılır. Her sürümde farklı syntax kullanılır:

      • Components (1.20.5+): [custom_name='...',enchantments={...}]
      • Modern (1.13-1.20.4): {display:{Name:'...'},Enchantments:[...]}
      • Legacy (1.8-1.12): {display:{Name:...},ench:[...]}

      Popüler item özellikleri: enchantment, custom name, lore, unbreakable, hide flags, can place on, can destroy, attribute modifiers.

      minecraft item komutu minecraft özel eşya minecraft büyülü item minecraft nbt item minecraft components item

      Mob Komutları

      Özel moblar oluşturmak için /summon komutu kullanılır. Boss moblar, özel özellikli hayvanlar ve NPC'ler oluşturulabilir.

      Kullanılan NBT tagleri:

      • CustomName - Özel isim
      • Health - Can puanı
      • Attributes - Özellikler (hız, hasar, boyut)
      • HandItems - Elindeki itemlar
      • ArmorItems - Zırhlar
      • ActiveEffects - Aktif efektler
      • NoAI - Yapay zeka kapama
      • Invulnerable - Hasar almama
      minecraft mob komutu minecraft boss oluşturma minecraft summon nbt minecraft özel canlı minecraft npc oluşturma

      Redstone ve Mekanizmalar

      Command blocklar ile karmaşık mekanizmalar oluşturulabilir. Command block türleri:

      • Impulse: Tek seferlik çalışır
      • Repeat: Her tick'te çalışır
      • Chain: Zincirleme çalışır

      Command blocklar ile yapılabilenler: otomatik kapılar, tuzaklar, mini oyunlar, mağaza sistemleri, görev sistemleri, boss savaşları.

      Dünya ve Oyun Ayarları

      Oyun dünyasını ve kurallarını değiştirmek için kullanılan komutlar:

      • /gamerule keepInventory true - Ölünce item kaybetmeme
      • /gamerule doDaylightCycle false - Zamanı durdurma
      • /gamerule doMobSpawning false - Mob doğmasını kapatma
      • /gamerule commandBlockOutput false - Command block mesajlarını kapatma
      • /difficulty peaceful - Barışçıl mod
      • /weather clear 999999 - Hava durumu sabitleme
      • /time set day - Gündüz yapma

      Sık Sorulan Sorular

      Minecraft komutları nerede yazılır?

      Minecraft komutları sohbet penceresinde (T tuşu) veya command blocklarda yazılır. Command blocklar creative modda /give @s command_block komutu ile alınabilir. Tek oyunculu dünyalarda hileler açık olmalıdır.

      Minecraft 1.21 ile 1.20 arasındaki komut farkı nedir?

      1.20.5 sürümünden itibaren Minecraft item komutlarında NBT tagleri kaldırıldı. Yerine [component=değer] sistemi getirildi. Bu değişiklik sadece item komutlarını etkiler, diğer komutlar (summon, execute vb.) aynı kalır.

      Command block nasıl alınır?

      Command block almak için sohbete /give @s command_block yazın. Command blocklar sadece creative modda ve op yetkisi olan oyuncular tarafından yerleştirilebilir. Survival modda command block kullanılamaz.

      Minecraft komutları çalışmıyor ne yapmalıyım?

      Komutların çalışması için: 1) Hileler açık olmalı, 2) Op yetkisi gerekebilir, 3) Komut doğru syntax ile yazılmalı, 4) Sürüm uyumlu olmalı. Hata mesajlarını dikkatle okuyun, genellikle sorunun ne olduğunu söyler.

      Datapack nasıl yüklenir?

      Datapack'ler dünya klasörünün içindeki datapacks klasörüne atılır. Oyun içinde /datapack list komutu ile görüntülenebilir ve /datapack enable "pack_ismi" ile etkinleştirilebilir.

      Minecraft komut bloğu zinciri nasıl kurulur?

      Command block zinciri kurmak için: 1) Repeat command block koyun, 2) Chain command blockları ardına yerleştirin, 3) Her blockun yönünü ok işareti ile bir sonrakine doğru çevirin, 4) Repeat blocka redstone sinyali verin. Zincir otomatik olarak çalışacaktır.
